## Dev Setup

```shell
# install deps
$ npm i

# run development
$ npm run dev
```

## Build Setup

```shell
# install deps
$ npm i

# run build
$ npm run build

# start
$ npm run start
```

For detailed explanation on how Nuxt.js works, checkout the [Nuxt.js docs](https://nuxtjs.org) and for the TailwindCSS check out the [TailwindCSS docs](https://tailwindcss.com/docs/installation).
